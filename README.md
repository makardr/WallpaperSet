# Wallpaper Crop

A lightweight Android app for cropping wallpapers — useful as a fallback when your device's native wallpaper picker is limited or unavailable.

## Features

- **In-app media browser** — pick any image from your device's gallery directly inside the app
- **Share target** — share an image from any app (gallery, browser, files) straight into this app
- **Set as-is** — apply the selected image as your wallpaper in one tap
- **Crop before setting** — use the built-in crop tool to fit the image perfectly to your screen
